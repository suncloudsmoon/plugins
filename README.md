# suncloudsmoon — Claude Code plugins

A marketplace of Claude Code plugins.

## Add the marketplace

```bash
claude plugin marketplace add suncloudsmoon/plugins
```

## Plugins

| Plugin | Install | What it does |
|--------|---------|--------------|
| **company-replicator** | `claude plugin install company-replicator@suncloudsmoon` | Research a company and generate a cited persona skill bundle that makes Claude operate as that company. |
| **stories** | `claude plugin install stories@suncloudsmoon` | A soul-bearing wiki for your codebase — read the stories before non-trivial changes, rewrite them after. |
| **chatgpt-deep-research** | `claude plugin install chatgpt-deep-research@suncloudsmoon` | Bridge Claude and ChatGPT Deep Research: craft a prompt, then ingest the returned PDF. |
| **prior-art** | `claude plugin install prior-art@suncloudsmoon` | Harvest reference UI into a gitignored `.ref/` swipe file and research best-known methods before building — copy what works. |

Restart Claude Code after installing.
